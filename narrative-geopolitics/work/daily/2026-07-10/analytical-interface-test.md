# When Better Labels Prevent a False Promotion

Status: `completed method comparison`

Date: `2026-07-11`

## Test

Compare the July 10 `best-intake-prep` scaffold with a completed run using:

`bounded crisis object -> mechanism-bearing lead judgment -> causal forecast review -> uncertainty with a named cause`

## Baseline

The prep scaffold contained five routed source placeholders but no claims, owning object, judgment, or forecast decision. Its structure invited two topic-level summaries: renewed Hormuz war and NATO summit escalation.

## Result

The analytical-interface method changed the decision in three ways:

1. It selected the alleged Hormuz bypass as a discriminating mechanism rather than treating source volume or rhetorical intensity as importance.
2. It refused to promote Russia–NATO because the batch lacked the deployable-capacity observable named by the watch object.
3. It refused to add a forecast because the apparent event duplicated an existing causal wager and remained operationally unverified.

| Bounded object | None | Permission-dependent Hormuz passage | Yes |

| Uncertainty cause | Generic prompt | Four named evidentiary causes | Yes |
| Watch promotion discipline | Not adjudicated | NATO remains watch candidate for lack of capability evidence | Yes |

## Falsifier

The method would fail this test if its more elaborate labels merely restated source rhetoric, forced a new forecast, or promoted NATO–Russia without a new observable. It did none of those.

## Remaining Limit

This is one same-day application, not a measured estimate of forecast improvement. It demonstrates better decision structure and restraint; it does not establish better geopolitical accuracy.

## Decision

`continue`

Inherit the four-part analytical chain for completed daily synthesis, while keeping source verification and forecast resolution as separate gates.
