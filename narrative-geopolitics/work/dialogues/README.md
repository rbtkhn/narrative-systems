# Council Dialogues

This directory holds manual experiments in corpus-grounded dialogue with the
project's curated voices.

Use [_template.md](_template.md). A dialogue is eligible for public promotion
only after it passes the fidelity review in
[the dialogue contract](../../method/dialogue-contract.md). Until then it
remains internal work and has no authority over archive sources, voice records,
forecast entries, or daily synthesis.

The working order is:

```text
question and corpus boundary
-> independent voice reconstructions
-> explicit challenges and disagreements
-> separate moderator synthesis
-> fidelity review
-> decision: continue, narrow, redesign, or stop
```

Automation is intentionally deferred. Manual work must first show which steps
are repetitive and which require irreducible judgment.
