# Predictive History System Model

## System Class

Predictive History is a public corpus traversal system.

Its primary job is to guide readers and LLMs through historical, literary, civilizational, and geopolitical source material without collapsing the layers of evidence, route, commentary, and interpretation.

## Layer Model

```text
doorway
-> catalog
-> route
-> source floor
-> commentary canvas
-> widened interpretation
```

## Contrast With Narrative Geopolitics

Predictive History begins from durable historical source material and builds a guided reading model.

Narrative Geopolitics begins from live transcript intake and builds a public geopolitical judgment model.
