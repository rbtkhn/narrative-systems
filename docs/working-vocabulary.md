# Working Vocabulary

These are provisional terms for the first scaffold. They are useful because they give experiments something concrete to name.

## Narrative Unit

A narrative unit is a named unit of meaning, action, or event. It can be small, like an image or line of dialogue, or larger, like a scene-level turn.

## Narrative Relation

A narrative relation is a typed connection between two units. It says that one unit changes, frames, enables, contests, echoes, or otherwise affects another.

## Narrative System

A narrative system is a collection of units and relations that can be inspected as a whole. In this scaffold, the system only stores structure and reports a small summary.

## Open Questions

- Which relation types become stable enough to model directly?
- When should a unit contain metadata beyond a name and note?
- What counts as evidence that a system has changed rather than merely grown?
